        AdventureWorks Sales Dashboard

Overview
Excel dashboard analyzing sales data from AdventureWorks2022, focusing on revenue, customers, products, and regional performance.

Tools
SQL Server
Python (pandas, pyodbc)
Excel (PivotTables, Charts)

Key Insights
Generated over $20M across 10.5K customers.
Maintained a 7-day average order fulfillment time.
Revenue is concentrated in a few top cities.
Certain product categories drive most sales.
